# 画像アセット・ディレクトリ (`images/`)

このディレクトリは、プロジェクトに関連する画像ファイルを保管するための場所です。

## 主な用途
* **ドキュメント用:** `README.md` やプロジェクトの解説書で使用するスクリーンショットや図解。
* **回路図・設計図:** プロジェクトの配線図（Fritzing, KiCadなど）の書き出し画像。
* **ロゴ・アイコン:** プロジェクトのシンボルマーク。

## 推奨される運用
* **フォーマット:** Web上での表示に適した `PNG`, `JPG`, `SVG`, `GIF` などを推奨します。
* **パスの指定:** プロジェクトのルートにある `README.md` から画像を参照する場合は、相対パスを使用してください。
  * 例: `![回路図](./images/schematic.png)`
* **軽量化:** リポジトリのサイズ肥大化を防ぐため、可能な限り画像ファイルは最適化（圧縮）してから保存することを推奨します。

---

> [!TIP]
> プログラム内で使用する「バイナリデータとしての画像（アイコンデータ等）」は、ここではなく `src/` や `include/` 内にヘッダーファイルとして配置するか、`data/` ディレクトリ（SPIFFS/LittleFS用）に配置するのが一般的です。

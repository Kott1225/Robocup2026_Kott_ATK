# ソースコード・ディレクトリ (`src/`)

このディレクトリは、プロジェクトの**メインソースコード**を配置するための場所です。
また，`include/`内にある**ヘッダファイルの詳細な記述**もこのディレクトリ内にあります．

## 基本構成
PlatformIO は、このディレクトリ内にあるファイルをコンパイルの対象とします。

* **`main.cpp`**: プログラムのエントリポイント（開始点）となるファイルです。
* **サブファイル**: コードの規模が巨大となってしまったため，機能クラスごとに分けて配置しています．

## ファイルの役割
* **`.cpp`ファイル**: 実際の処理内容（関数の実装など）を記述します。
* **`#include` について**: 
  * プロジェクト共通のヘッダーは `../include` から読み込みます。
  * プライベートなライブラリは `../lib` から読み込まれます。

## `main.cpp` の基本構造
```cpp
#include <Arduino.h> // Arduinoフレームワークの場合

void setup() {
  // 初期化処理
}

void loop() {
  // メインループ
}
```
